## Getting Started

### Prerequisites

- [Python][python-url]
- A CSV dataset containing just two fields: `job_id` and `description`.


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Pobl-Group/mould-analysis.git
   ```
1. Install packages
   ```sh
   pip install -r requirements.txt
   ```
